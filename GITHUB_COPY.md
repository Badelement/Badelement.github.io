### GitHub About

An Apple-style GitHub homepage that turns a small set of repositories into a calm, polished personal portfolio.

This project is the front door to my GitHub.

Instead of listing repositories in a plain grid, it presents them with a softer, more intentional Apple-style visual language. It is meant to feel light, clear, and personal, while still staying simple enough to maintain as I keep learning and building.

## 2. doc_translation_tool

### GitHub About

A desktop tool for translating Markdown documents between Chinese and English while preserving structure, formatting, and flow.

### README Opening

Document Translation Tool is a desktop utility built for a simple but practical idea: translate Markdown documents without breaking the structure that makes them usable.

It focuses on keeping the flow calm and predictable. The document stays intact, the interface stays clear, and the translation process feels more like using a tool than wrestling with raw model output.

### Homepage Card Copy

- `tagline`
  Markdown 文档翻译桌面工具
- `intro`
  这是一个更偏实用型的项目。我想把文档翻译这件事做得更安静、更顺滑一些，让 Markdown 结构、内容和输出结果都保持清晰。

## 3. openclaw-stack

### GitHub About

A workflow-focused OpenClaw skill stack for turning scattered AI tasks into a more structured local engineering system.

### README Opening

OpenClaw Stack is an experiment in structure.

Instead of treating AI as a single assistant with a long prompt, this project explores a calmer and more modular workflow: product thinking, architecture, review, testing, and orchestration, all arranged as a more intentional system.

### Homepage Card Copy

- `tagline`
  OpenClaw 技能包与工作流集合
- `intro`
  这是一个偏实验性的方向。我想把零散的 AI 工具和技能整理成更有秩序的工作流，让整个过程看起来更像系统，而不只是零碎命令的堆叠。
